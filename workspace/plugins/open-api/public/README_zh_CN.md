# 开放 API

将插件的 API 公开至全局变量 `globalThis.openAPI` / `window.openAPI` / `openAPI` 以帮助开发者测试 APIs.

## 预览

![预览图片](./preview.png)

## 介绍

- `openAPI`: 全局变量
  - `plugin`: 插件对象
  - `siyuan`: 思源的插件 API 接口
